# AI Coding Assistant Project Pack

This pack contains planning and specification documents for the group project: a command-line autonomous AI coding assistant with MCP integration, provider abstraction, and a custom RAG server.

Use these docs as the planning baseline before implementation. They are written so a coding agent can execute against them with minimal ambiguity.

## Included files

- `01_project_scope_and_requirements.md`
- `02_system_architecture.md`
- `03_repo_structure.md`
- `04_state_diagram.md`
- `05_sequence_diagrams.md`
- `06_module_specs_cli.md`
- `07_module_specs_agent_loop.md`
- `08_module_specs_provider_abstraction.md`
- `09_module_specs_mcp_client.md`
- `10_module_specs_rag_server.md`
- `11_integration_plan_and_task_split.md`
- `12_definition_of_done.md`

## Suggested order of use

1. Read project scope
2. Read architecture
3. Create repo using repo structure
4. Implement module specs in this order:
   - CLI
   - provider abstraction
   - agent loop
   - MCP client
   - RAG server
5. Follow integration plan
6. Validate against definition of done
